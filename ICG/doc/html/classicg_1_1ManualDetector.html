Detailed Description

Detector that allows a user to select 4 points in the image and that, based on those points, infers the pose of a referenced Body object.

To select points, the PointDetector class is employed. Using the main method DetectBody(), the detection process is started and the pose of the referenced body is set.

Parameters
body_ptrreferenced Body object for which the pose is set.
color_camera_ptrreferenced ColorCamera object that is used.
reference_points4 3D model points on the body's surface that are given in the body reference frame.
detector_image_pathoptional path to a helper image in which the position of reference_points can be marked.
